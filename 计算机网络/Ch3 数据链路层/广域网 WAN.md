- 广域网的通信子网使用传输方式为**存储转发式**的**分组交换技术**
- 广域网的拓扑结构通常采用**网状**
- [[路由器 Router]]实现[[局域网 LAN]]和广域网的互联

## 广域网和[[局域网 LAN]]的区别与联系

|            | 广域网           | 局域网          |
| ---------- | ------------- | ------------ |
| 覆盖范围       | 很广，通常跨区域      | 较小，通常在一个区域内  |
| 连接方式       | 通常采用**点对点连接** | 普遍使用**广播信道** |
| OSI 参考模型层次 | 前三层           | 前两层          |
| 着重点        | 强调**资源共享**    | 强调**数据传输**   |

二者联系：两者平等，不是包含关系；当连接到一个广域网或一个局域网上的主机在该网内进行通信时，只需要使用其网络的物理地址。

## 点对点协议 PPP

| 三个组成部分               | 功能                                              |
| -------------------- | ----------------------------------------------- |
| 一个链路控制协议（LCP）        | **LCP 用来建立，配置和测试数据链路连接的协议**。双方可以协商一些选项。         |
| 一套网络控制协议（NCP）        | **NCP 用来协商不同的网络协议**，它包含了众多协议，因此它对其它很多协议都具备协商功能。 |
| 一种将 IP 数据包封装到串行链路的方法 | 支持**异步链路和同步链路**。                                |

### 不同传输方式的对比

| 传输方式 | 实现透明传输的方法                                        |
| ---- | ------------------------------------------------ |
| 同步传输 | [[封装成帧#零比特填充法\|零比特填充法]]                          |
| 异步传输 | [[封装成帧#字节填充法\|字节填充法]]，使用的转义字符为 `0x7D (01111101)` |

### PPP 协议的特点

1. PPP 是**面向字节**的
2. **不可靠服务**，只保证无差错接收（**CRC**），不使用序号和确认机制
3. PPP 的**两端可以运行不同的网络层协议**
4. PPP 只支持**全双工通信**
