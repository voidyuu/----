网络地址转换是指通过将专用网络地址转换为**公用地址**，从而对外隐藏内部管理的 IP 地址。因为专用网本地 IP 地址是可重用的，所以 NAT 大大**节省了 IP 地址的消耗**。同时它隐藏了内部网络结构，从而降低了内部网络受到攻击的风险。

NAT 转换表中存放{本地 IP 地址：端口}到{全球 IP 地址：端口}的映射，可以将多个私有地址映射到一个全球 IP 地址。

## 私有 IP 地址网段

| 网络类型 | 网段数量  | 范围                                    |
| ---- | ----- | ------------------------------------- |
| A    | 1 个   | **10**. 0.0.0~**10**. 255.255.255     |
| B    | 16 个  | **172.16**. 0.0~**172.31**. 255.255   |
| C    | 256 个 | **192.168**. 0.0~**192.168**. 255.255 |
