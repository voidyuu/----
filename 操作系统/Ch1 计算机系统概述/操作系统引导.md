1. 激活 CPU。激活的 CPU 读取 [[只读存储器 ROM|ROM]] 的 bootstrap 程序（**自举程序**），将 IR 置为 BIOS 的第一条指令，即开始执行 BIOS 的指令
2. 硬件自检
3. 加载带有操作系统的硬盘。通电自检后，BIOS 开始读取 Boot Sequence（通过 CMOS 里保存的启动顺序，或与用户交互的方式），选定第一个存储设备，给予控制权。
4. 加载 [[文件系统布局#主引导记录 MBR|主引导记录 MBR]]
5. 扫描硬盘分区表
6. 加载[[文件系统布局#引导块 Boot Block|分区引导记录 PBR]] 。读取**活动分区**的 0 号扇区，这个扇区称为**引导扇区**，其作用是寻找并激活分区根目录下用于引导操作系统的程序（启动管理器）
7. 加载启动管理器（如 Windows Boot Manager）
8. 加载操作系统

> [!tip]
> BIOS程序在内存最开始的空间构建中断向量表和相应服务程序。
