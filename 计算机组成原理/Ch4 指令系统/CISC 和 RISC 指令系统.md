|           | CISC  | RISC          |
| --------- | ----- | ------------- |
| 指令系统      | 复杂，庞大 | 简单，精简         |
| 指令数量      | 多     | 少             |
| 指令字长      | 不固定   | 固定            |
| 访存指令      | 不固定   | 只有 LOAD/STORE |
| 各种指令执行时间  | 相差很大  | 大多数都在一个周期内完成  |
| 各种指令使用频率  | 频度相差大 | 比较平均          |
| 通用寄存器数量   | 少     | 多             |
| 控制方式      | 微程序控制 | 硬布线控制         |
| [[指令流水线]] | 可选    | 一定采用          |
| 典型架构      | x86   | ARM、MIPS 架构   |
