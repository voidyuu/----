源操作数：用于参与计算。
目的操作数：用于参与计算并保存计算结果。

>[!tip]
>有时候题目指明了目的操作数，不一定是保存到 A1，以题目为准。

| 指令类型  | 含义                                       |
| ----- | ---------------------------------------- |
| 零地址指令 | 不需要操作数的指令或操作数隐含的指令如：空操作指令 nop、停机指令、关中断指令 |
| 一地址指令 | OP(A1)->A1 或  (ACC)OP(A1)->ACC           |
| 二地址指令 | (A1)OP(A2)->A1                           |
| 三地址指令 | (A1)OP(A2)->A3                           |
| 四地址指令 | (A1)OP(A2)->A3，A4 为下一条指令地址               |
