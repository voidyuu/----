## 存储芯片的片选

### 线选法

n 个芯片需要 n 位片选位。

| 芯片  | A14​∼A11​ |
| :-: | :-------: |
| #0  |   1110    |
| #1  |   1101    |
| #2  |   1011    |
| #3  |   0111    |

### 译码片选法

使用一块 74LS138 芯片作为地址译码器。n 个芯片需要 logn 位片选位。
